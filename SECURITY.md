# Security Policy

## Supported Versions

This is a static site project. Security fixes will generally be applied to the latest default branch.

## Reporting a Vulnerability

Please do **not** open public issues for sensitive vulnerabilities.

Instead, report privately to the maintainer with:

- Affected file(s)
- Reproduction steps
- Impact assessment
- Suggested mitigation (if available)

We will acknowledge receipt and provide a follow-up plan as soon as possible.
