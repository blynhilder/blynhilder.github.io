# Claude 参考版本审阅

## 逐屏观察

1. **首页首屏**：黑色液态岛是最明确的视觉锚点，状态感比普通整宽导航更强；应保留。
2. **控制中心**：厚玻璃、抓手、六宫格和背景去焦构成完整交互状态，是参考版本最成熟的部分；已按本站导航重建。
3. **材质图鉴**：用三个模糊等级解释玻璃层级很有效，但参考版本进入该区前留白偏长；实现保留概念并压缩节奏。
4. **弹簧实验**：把“苹果感”从形容词变成可调参数，兼具展示与教学价值；实现加入真实阻尼振子和运行读数。
5. **专题文章**：内容长度足以支撑继续探索，相邻导航也很自然；实现保留五篇长读，同时补充来源、回到主站路径和主题页入口。

## 总体健康度

- **视觉语言**：参考版本的黑色状态岛与白色厚玻璃面板关系清晰，适合成为全站固定导航。
- **信息架构**：参考版本偏向独立展示；融合后变成“首页总览 → 五大主题 → 四条分支 → 专题长读”的完整层级。
- **动效**：保留弹性出现、背景去焦、拖拽关闭和参数实验；同时支持 `prefers-reduced-motion`。
- **可访问性**：实现补充对话框语义、焦点恢复、隐藏面板 `inert`、控件名称和键盘关闭。
- **内容可信度**：新增来源卡片把论文、官方说明与原始文本放到文章末尾，读者可以继续核验。

结论：参考版本的液态岛和交互实验值得完整吸收；在本站中最合适的升级方式，是把它们变成贯穿既有知识架构的导航与学习工具，而不是孤立的视觉样品。
