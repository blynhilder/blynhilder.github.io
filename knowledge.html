1) C++ 是什么？

C++ 是一种高性能通用语言，广泛用于系统、游戏、图形、嵌入式等方向。建议从语法基础 -> STL -> 面向对象 -> 现代 C++（C++11/14/17/20）逐步推进。

2) 变量与基本类型

常见类型：int、double、char、bool、std::string。尽量在定义时初始化，使用 const 表示不可修改。

#include <iostream>
#include <string>
int main() {
  int age = 18;
  const double pi = 3.14159;
  std::string name = "Alice";
  std::cout << name << " " << age << " " << pi << "\n";
}

3) 条件与循环

分支：if/else、switch；循环：for、while。建议先写清楚业务条件，再优化结构。

4) 函数与参数传递

函数有返回值与参数列表。值传递会复制，引用传递更高效；只读参数可用 const &。

int add(int a, int b) { return a + b; }
void printName(const std::string& name) {
  std::cout << name << "\n";
}

5) STL 入门：vector

std::vector 是动态数组，支持自动扩容。

#include <vector>
std::vector<int> nums{1,2,3};
nums.push_back(4);
for (int x : nums) {
  // use x
}

6) 内存管理与 RAII

RAII 是现代 C++ 核心思想。优先使用对象和智能指针管理资源，减少手动 new/delete 的风险。
