# 个人索引库（GitHub 新手友好版）

这是一个**可以直接开始用**的个人知识索引库模板：
- 不需要复杂技术；
- 先记录，再整理；
- 适合“很懒但想持续积累”的使用方式。

## 你现在可以怎么用

### 1) 最简单（每天 1 分钟）
1. 在 `daily/` 上传一张当天笔记照片（或一段文字）。
2. 文件名用日期：`YYYY-MM-DD-主题.md` 或 `YYYY-MM-DD-主题.jpg`。
3. 每周把有价值内容整理到 `notes/`。

### 2) 略进阶（每周 10 分钟）
1. 新建一篇 `notes/` 笔记（用模板）。
2. 在 `INDEX.md` 里补一行索引。
3. 给仓库设置 Topics（如：`knowledge-base`、`notes`、`learning`）。
